# Network Penetration Testing Assignment

This repository contains the complete documentation, reports, and resources for the Network Security and Penetration Testing CA1 assignment at National College of Ireland.

- **Module**: Network Security and Penetration Testing
- **CA Type**: Group Assessment (CA1)
- **Weight**: 40% of overall module marks
- **Due Date**: Friday 14th November 2025 @ 23:55

## Quick Start Guide

### 1. Read the Assignment Brief
- Review all requirements carefully
- Understand the marking scheme
- Note submission deadlines

### 2. Follow PROJECT_PLAN.md
- This contains the complete step-by-step guide
- Follow phases sequentially
- Document everything as you go

### 3. Research Platforms
- Use `REFERENCES/platforms_research_template.md`
- Compare at least 5-6 platforms
- Select 4 systems (3 new + 1 optional custom)

### 4. Set Up Environment
- Install Kali Linux
- Set up VPN access (if needed)
- Install required tools
- Create documentation structure

### 5. Conduct Penetration Tests
- Follow PTES methodology (see METHODOLOGY.md)
- Use `DOCUMENTATION/system_template.md` for each system
- Document everything: commands, outputs, screenshots

### 6. Write Report
- Follow report structure in PROJECT_PLAN.md
- Use IEEE citation style
- Target ~25 pages (main content)

### 7. Create Presentation
- 10-15 slides
- 15-minute presentation (10-12 min + Q&A)
- Practice timing

### 8. Complete Forms
- AI Acknowledgement Form
- Peer Evaluation Forms (one per student)

## Key Requirements

### Systems Selection
- ✅ 4 systems total
- ✅ At least 3 released in past 3 months
- ✅ Systems must complement each other
- ✅ Different OS, services, vulnerabilities
- ✅ Complete network penetration test possible
- ❌ Cannot use Metasploitable, DVWA, or lab systems

### Report Requirements
- ✅ ~25 pages (main content, excluding appendices)
- ✅ Single unified report (not separate reports)
- ✅ IEEE citation style
- ✅ Professional formatting
- ✅ Detailed appendices for reproduction

### Presentation Requirements
- ✅ 10-15 slides
- ✅ 15-minute timeslot (10-12 min presentation + 3-5 min Q&A)
- ✅ All group members must attend

## Important Reminders

- ✅ Properly cite all sources (IEEE style)
